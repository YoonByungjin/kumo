☁️ 프로젝트 요약: Kumo (일본 아르바이트 구인/구직 플랫폼)

1. 프로젝트 개요
- 프로젝트 명: Kumo (くも)
- 개발 기간: 2026.02 ~ 2026.03 (2개월)
- 팀 인원수: 4명
- 프로젝트 설명: 일본 주요 도시(도쿄, 오사카 등)를 대상으로 하는 지도 기반의 아르바이트 구인/구직 서비스입니다. 다국어(한/일) 지원과 실시간 소통 기능을 통해 한일 양국 사용자에게 최적화된 구인 환경을
  제공합니다.

2. 본인 담당 역할 (Core Developer / Full-stack)
   비로그인 사용자 경험(UX)부터 구직자 핵심 기능, 시스템 보안 및 백엔드 인증 아키텍처 전반을 전담하여 개발했습니다.

🛡️ Security & Authentication (전담)
- 보안 아키텍처 설계: Spring Security를 활용한 시스템 전반의 인증/인가 체계 구축
- 로그인 최적화: Ajax 인증 성공/실패 핸들러 구현 및 로그인 시도 제한 로직 개발
- 안정성 강화: Google reCAPTCHA 필터 도입 및 보안 예외 처리 총괄

💻 Frontend Development
- NonLoginView (100%): 메인 홈(home.html), 로그인, 회원가입(구직자/구인자), 아이디/비밀번호 찾기 등 서비스 진입점의 모든 페이지 개발
- SeekerView (100%): 구직자 마이페이지, 이력서 작성/관리, 지원 현황, 스카우트 제안 관리, 프로필 수정 등 구직자 전용 기능 전체 개발
- AdminView: 관리자용 시스템 로그 출력 및 모니터링 페이지 구현

⚙️ Backend Development (핵심 컨트롤러 전담)
- 인증 및 계정 관리: LoginController(로그인/로그아웃), MailController(이메일 인증 및 비번 찾기)
- 구직자 비즈니스 로직: SeekerController(이력서, 프로필, 마이페이지 처리)
- 실시간 알림 시스템: NotificationController(SSE/WebSocket 기반 실시간 알림 로직)
- 공통 및 API 설계: HomeController(메인 라우팅), ApiController(파일 업로드 및 공통 API 처리)

3. 주요 기능
- 통합 인증 시스템: 구직자/구인자 맞춤형 회원가입 및 이메일 기반 보안 인증
- 스마트 이력서 관리: 상세 경력, 자격증, 어학 능력을 관리하고 기업의 스카우트 제안을 받는 기능
- 실시간 소통: 구인자와의 실시간 채팅 및 상태 변화에 따른 즉각적인 알림 서비스
- 시스템 모니터링: 관리자를 위한 실시간 서비스 로그 추적 기능

4. 사용 기술 스택
- Language: Java 21, JavaScript
- Framework: Spring Boot 3.5.8, Spring Security
- View Engine: Thymeleaf
- Database: MySQL, Spring Data JPA
- Library: jQuery 3.7.1, WebSocket, Spring Mail
- Testing: Playwright (E2E)
